# Mp3-Duration - Get the duration of an MP3 file

## Usage

```common-lisp
(import 'mp3-duration:estimate-duration)

;; Get the duration of an MP3 file in seconds.
(estimate-duration #P"Music/ピノキオP/ありふれたせかいせいふく.mp3")
;=> 202.97144
```
